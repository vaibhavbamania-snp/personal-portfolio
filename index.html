🚀 Curiosity Driven Projects
Minecraft Elytra Automation GitHub ↗
Python · Empirical Modelling · Curve Fitting · Dynamic Error Correction

Manually collected data from 400+ flights — recording velocity components (vx, vy) across angles — then applied curve fitting to derive predictive functions for flight behaviour. Built a full prediction engine that outputs required launch angle, height, and exact landing time. Tackled non-uniform velocity (the acceleration phase before terminal velocity) which caused significant average shifts on shorter flights — modelled this separately and blended it into the prediction. Designed a dynamic error compensation channel that tracks and corrects accumulating drift as a function within the code. Final accuracy: 99.9984%+ — under 7–8 blocks of error over 150,000 block flights. Built entirely solo, zero AI assistance.

Marble Interception System
Classical Mechanics · DC Motor Dynamics · Deterministic Systems

Built a physical system that intercepts a marble rolling down an incline — using zero external sensors or visual tracking. Modelled the target marble's incline dynamics and timing from pure kinematics. The interceptor is fired by a DC motor, with full mathematical compensation for motor acceleration profile and electrical trigger delay (the gap between current reaching the coil and the motor actually moving). Every variable accounted for in equations. The result: a deterministic, open-loop interception system that works purely on math — and it hits.
